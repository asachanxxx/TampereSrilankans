## To replace with your actual images:

### 1. Hero Background (Top Section)
- **File name:** `hero-background.jpg` (or `.png`, `.webp`)
- **Recommended size:** 1920 × 1080 pixels (16:9)
- **Alternative sizes:** 2560 × 1440, 1600 × 900
- **Purpose:** Background for "Discover Amazing Events" section
- **Note:** Image will have an overlay for text readability

### 2. About Section
- **File name:** `about-section.jpg` (already added ✓)
- **Size:** 1920 × 1080 pixels (16:9)

## How to replace:

1. Add your image file to this folder
2. Name it exactly as shown above
3. Refresh your browser (Ctrl + Shift + R)

## How to reference images in Next.js:

- Place images in this `public/images/` folder
- Reference them in code with `/images/filename.ext`
- Next.js will automatically serve them

Example:
- File location: `public/images/my-photo.jpg`
- In code: `<img src="/images/my-photo.jpg" alt="..." />`
